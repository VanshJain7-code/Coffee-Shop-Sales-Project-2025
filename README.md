# Coffee-Shop-Sales-Project-2025
This is an Excel-based data analysis project that explores and visualizes sales performance for a coffee shop. The project uses Power Query, Pivot Tables, and interactive Excel dashboards to show insights about product sales, revenue trends, and customer behavior.

Project Overview:
The Coffee Sales Shop Dashboard provides a clear and interactive way to analyze sales data. It helps identify which coffee products generate the most revenue, analyze sales trends over time, and compare performance across regions, product types, and customer segments.

Tools and Technologies Used:

Microsoft Excel

Power Query for data cleaning and transformation

Pivot Tables and Charts

Conditional Formatting

Dashboard design using slicers and visuals
